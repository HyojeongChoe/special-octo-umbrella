git 

1. 로컬저장소(git.test폴더) - 원격저장소(내 리포지토리) 연결

- git init -> 만들어두고 시작
- git clone [원격저장소 주소] ->코드 들어가서 복사

-저장소 안에 들어가서 bash 열기

-	git add . 	"모든 파일에 대해 작업한다."
2. commit	변경된 사항을 기록
	git commit -m " 커밋메시지"
3. push		로컬->원격 자료 이동
		커밋 한 다음에 푸시해야함
	git push [원격지주소] main
4. pull		로컬<-원격 당겨받기
		단독으로 당겨받음
	git pull [원격지주소] main
