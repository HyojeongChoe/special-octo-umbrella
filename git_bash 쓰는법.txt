git 

1. 로컬저장소(git.test폴더) - 원격저장소(내 리포지토리) 연결

- git init -> 만들어두고 시작
- git clone [원격저장소 주소] ->코드 들어가서 복사

-저장소 안에 들어가서 bash 열기

-	git add . 	"모든 파일에 대해 작업한다."
2. commit	변경된 사항을 기록
	git commit -m " 커밋메시지"
3. push		로컬->원격 자료 이동
		커밋 한 다음에 푸시해야함
	git push [원격지주소] main
4. pull		로컬<-원격 당겨받기
		단독으로 당겨받음
	git pull [원격지주소] main


 *브랜치(branch): "나뭇가지" -> 버전
- main : 버전의 "중심" / 실제
- main : 이외의 버전 : 테스트용
	->사용자 편의대로 이름 설정

(master) 뜨면 main으로 바꾸기
git branch -m master main

5. 브랜치 생성: git.branch [생성할 브랜치명]

6. 작업할 브랜치 변경 : git.
